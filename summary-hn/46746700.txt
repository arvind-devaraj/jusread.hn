Show HN: Polymcp – Turn Any Python Function into an MCP Tool for AI Agents  (23 pts, 7 comments)
https://news.ycombinator.com/item?id=46746700

The Hacker News discussion centers around Polymcp, a framework that enables users to convert any Python function into a Model Context Protocol (MCP) tool for AI agents without the need for complex integrations or rewrites. The post provides examples of how simple functions, API calls, and business workflows can be exposed as MCP tools, allowing AI agents to utilize them directly. Key benefits highlighted include the ability to reuse existing code, automate workflows, and reduce development time while ensuring reliability through built-in validation and error handling.

In the comments, there is a debate about Polymcp's advantages compared to existing tools like FastMCP. One commenter questions the need for Polymcp, suggesting that exposing Python functions as MCP servers is already straightforward with FastMCP. The creator of Polymcp responds by emphasizing its additional features, such as unified orchestration across multiple MCP servers and a user interface for inspection. Another point of discussion involves concerns about handling side effects and security, with suggestions for prioritizing audit logs and short-lived credentials for tool calls. Overall, the conversation reflects a mix of curiosity about Polymcp's unique offerings and caution regarding its implementation in secure environments.
